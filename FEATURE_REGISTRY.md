## Database Admin Deep Workflows

- `backup-restore-checks` - Backup Restore Checks: Backup freshness, restore validation, retention gaps, and recovery readiness.
- `index-advisor` - Index Advisor: Index candidates, query patterns, cardinality, write impact, and tuning recommendations.
- `connection-pool-monitor` - Connection Pool Monitor: Pool saturation, wait times, idle sessions, timeouts, and capacity actions.
- `database-security-review` - Database Security Review: Roles, grants, exposed data, audit settings, and security remediation.

## Deep enhancement: Database Admin Deep Workflows

## Web Scraping Deep Workflows

- `robots-compliance-review` - Robots Compliance Review: Robots rules, allowed paths, crawl delays, source terms, and compliance status.
- `selector-drift-monitor` - Selector Drift Monitor: Selector failures, layout changes, fallback selectors, and repair priority.
- `proxy-retry-queue` - Proxy Retry Queue: Retry attempts, proxy health, throttling signals, and blocked source recovery.
- `scraped-data-quality` - Scraped Data Quality: Completeness, duplicates, schema mismatches, normalization, and quality scoring.

- `visual-qa-review` - Visual QA Review: Screenshots, layout issues, responsive defects, contrast notes, and bug priority.
